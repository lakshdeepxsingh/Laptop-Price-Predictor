# Laptop Price Predictor

A machine learning web application built with Streamlit that predicts laptop prices based on hardware specifications.

## Features

- Predict laptop prices instantly
- Trained using a machine learning regression model
- Interactive Streamlit interface

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Files

- `app.py` – Streamlit application
- `Laptop-Price-Predictor.ipynb` – Model training notebook
- `pipe.pkl` – Trained prediction pipeline
- `df.pkl` – Dataset used by the application
- `laptop_data.csv` – Original dataset

## Run Locally

```bash
streamlit run app.py
```
